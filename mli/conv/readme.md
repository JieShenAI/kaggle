# 卷积核

卷积可以由全连接变形过来

卷积核的大小不是越大越好:

&emsp;这个问题相当于问，MLP中隐藏层的大小不是越大越好。在MLP中窄一点深一点的网络比宽一点浅一点的网络要好。

卷积核，主流是(3,3)，最多是(5,5)

关于卷积核选择，Inception设计思路。不同尺寸的kernel进行计算，计算出一个合适的kernel。

# 局部性和平移不变性

卷积核体现局部性；无论目标在哪个地方都是同样的卷积核作用于它体现了**平移不变性**。


# 池化

池化不做多通道融合，卷积做多通道融合；

池化层的输出通道数 = 输入通道数

> 池化层没有输出通道的超参数

## 种类
* 最大池化层
* 平均池化层

池化总结
* 池化返回窗口中的最大值或平均值
* 缓解卷积层对位置的敏感性：**通常池化层作用在卷积层之后**
* 同样有窗口的大小、填充和步幅作为超参数
